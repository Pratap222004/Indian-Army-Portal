## 💻 Getting Started

```bash
# Clone the repo
git clone https://github.com/your-username/Indian_Army_Portal.git
cd Indian_Army_Portal

# Install frontend
cd client
npm install

# Install backend
cd ../server
npm install

# Run frontend
cd ../client
npm run dev

# Run backend
cd ../server
node index.js
